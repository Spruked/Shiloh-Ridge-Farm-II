# Shiloh Ridge Canonical Vault System

This is the repository's only vault. Every persistent runtime write must resolve beneath this directory.

The system is organized into auditable sub-vaults:

- `apriori/` and `posteriori/`: Renova canonical and learned cognition.
- `farm_orb/`: Orb Weaver crawls, pointer maps, and site-ORB context.
- `substrate/`: derived indexes, OCR, pointers, and human escalations.
- `shep/`: admin state, tool-call audit, worker learning, and traces.
- `butch/`: customer memory, orders, promotions, and voice cache.
- `documents/`, `nft/`, `validators/`, and `worker_chat/`: durable domain records.
- `audit/`: append-only events and the hashed file inventory.
- `backups/`: append-only Mongo and file snapshots.

No other directory named `vault` or `vault_system` is permitted in this repository.
